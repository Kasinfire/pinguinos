# Pingüinos del Mundo (Edición Retro)


## Sobre el Proyecto
Esta página web fue construida utilizando **100% HTML puro**. Cero CSS, cero JavaScript. Aquí usamos pura etiqueta loollllllll

## Especies en la Enciclopedia
* **Pingüino Emperador:** El gigante del hielo.
* **Pingüino de Adelia:** Pequeños, rebeldes y ladrones de piedras.
* **Pingüino de Galápagos:** Los únicos que se aguantan el calor.
* **Pingüino Azul:** Tamaño de bolsillo y nocturnos.
* **Pingüino Rey:** Los primos coloridos que no hacen nidos.
* **Pingüino de Barbijo:** Los del casquito de bicicleta.
